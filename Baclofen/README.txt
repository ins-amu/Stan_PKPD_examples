Code to fit a PK model to synthetic data with HMC-NUTS in Stan with the cmdstanpy package. 
We use two models: a model with informative priors on PK parameters and a model with wide priors.

Reproduce figures on synthetic data from Nina Baldy et al 2023 Mach. Learn.: Sci. Technol. 4 035048 (DOI 10.1088/2632-2153/acf6aa) https://iopscience.iop.org/article/10.1088/2632-2153/acf6aa
